Возможно, отдельные проекты (не форкнутые с учебного плана, а сделанные с нуля самостоятельно, позднее будут размещены здесь. Но это не точно).

Аргументы, которые побудили меня отказаться от публичности в данном вопросе, приведены здесь: https://habr.com/ru/articles/931286/
